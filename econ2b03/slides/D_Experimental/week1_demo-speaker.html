Quarto-Quiz (Ordering)

Order these steps for a basic hypothesis test.

  1. State the null and alternative hypotheses
  2. Choose a significance level
  3. Compute the test statistic
  4. Compare to a critical value (or compute a p-value)
  5. Draw a conclusion
