Part I: Random Variables

Concepts

  • A random variable is a numerical quantity that is generated by a random experiment.
  • We will abbreviate RV = random variable.
  • RVs assign a numerical value to any possible outcome of a random experiment.
  • Why is that useful?
    • RVs let us do arithmetic with randomness

Types of random variables

  • We will consider RVs where the set of possible values:
    • is finite, e.g. \(\{1,5,12\}\)
    • is infinite but countable, e.g. \(\{1,2,3,\cdots\}\)
    • includes a whole interval
  • A set of possible values is countable if all possible values can be listed one after another.
  • A discrete random variable has a finite or countable number of possible values.
    • this lecture focuses on discrete RVs
  • A continuous random variable has possible values that contain a whole interval of real numbers.

Discrete and continuous RVs

Fair coin as a random variable

  • Flip a coin. Assign \(X = 1\) for heads, \(X = 0\) for tails.
  • Because \(X \in \{0,1\}\), this is a discrete RV.

Dice roll outcomes

  • Roll a die. Let \(X\) be the number showing.
  • Because \(X \in \{1,2,3,4,5,6\}\), this is a discrete RV.

Goals in a hockey game

  • Let \(X\) be the number of goals scored by one team in a game.
  • Because \(X \in \{0,1,2,3,\ldots\}\), this is a discrete RV (countably infinite).

Courses taken per semester

  • Let \(X\) be the number of courses a randomly selected student takes this semester.
  • Because \(X \in \{1,2,3,4,5,6,7\}\), this is a discrete RV (finite).

Job applications before first offer

  • A graduate sends out applications one at a time. Let \(X\) be the number sent before receiving the first offer.
  • Because \(X \in \{1,2,3,\ldots\}\), this is a discrete RV (countably infinite).

Part II: Probability Distributions

Definition

  • The probability distribution of a discrete random variable \(X\) is a list of each possible value of \(X\) together with the probability that \(X\) takes that value in one trial of the experiment.
  • We will use the following notation:

    • \(x\) is a possible value of the discrete RV \(X\)

    • \(P(x)=P(X = x)\): prob that \(X\) takes value \(x\) in one trial

    • A distribution is a list of all possible values and their chances

Fair coin distribution

  • Flip a coin: \(X = 1\) for heads, \(X = 0\) for tails (discrete, finite).
\(x\) 0 1
\(P(x)\) 1/2 1/2

Unfair coin distribution

  • If heads comes up 80% of the time:
\(x\) 0 1
\(P(x)\) 1/5 4/5

Fair die distribution

  • Roll a die. Let \(X\) be the number showing (discrete, finite).
\(x\) 1 2 3 4 5 6
\(P(x)\) 1/6 1/6 1/6 1/6 1/6 1/6

Internship offers

  • A student applies to 4 internships, each with a 25% chance of an offer, independently. Let \(X\) = number of offers.
\(x\) 0 1 2 3 4
\(P(x)\) 0.3164 0.4219 0.2109 0.0469 0.0039
  • We will study this as a binomial distribution later.

Properties of probability distributions

  • A valid probability distribution must satisfy two conditions:
    1. each probability \(P(x)\) must be between 0 and 1: \(0\leq P(x)\leq 1\)
    2. the sum of all the probabilities is 1: \(\sum_x P(x)=1\)
  • Summation is over all possible values of the RV \(X\).

Part III: Mean and Variance

Mean of a discrete RV

  • The mean of a discrete random variable \(X\), also called the expected value, is the number \[\mu=E(X)=\sum_x xP(x).\]
  • Interpretation:
    • average across many repeated trials of the experiment
    • weighted average of values of \(X\), with probabilities as weights
    • the expected value is the long-run average

Help desk calls per hour

  • The number of calls to a help desk in an hour has the distribution:
\(x\) 0 1 2 3
\(P(x)\) 0.10 0.30 0.40 0.20
  • \(E(X)=0(0.10)+1(0.30)+2(0.40)+3(0.20)=1.7\)
  • About 1.7 calls per hour on average.

Fair coin expected value

  • Recall: \(P(X=0) = P(X=1) = 1/2\).
  • \(E(X) = 0 \times 1/2 + 1 \times 1/2 = 0.5\)

Unfair coin expected value

  • Recall: \(P(X=0) = 1/5\), \(P(X=1) = 4/5\).
  • \(E(X) = 0 \times 1/5 + 1 \times 4/5 = 4/5\)

Insurance payout

  • Suppose an insurance policy pays $0 with probability 0.95 and $10,000 with probability 0.05.

  • The expected payout is: \[E(X)=0\times0.95+10{,}000\times0.05=500.\]

  • Interpretation: the long-run average payout is $500 per policy.

Defective items in a shipment

  • A factory ships batches of 3 items. Each item is defective with probability 0.05, independently. Let \(X\) = number of defective items per batch.
\(x\) 0 1 2 3
\(P(x)\) 0.8574 0.1354 0.0071 0.0001
  • \(E(X) = 0(0.8574) + 1(0.1354) + 2(0.0071) + 3(0.0001) = 0.15\)
  • On average, 0.15 defective items per batch, or about 1 defective item per 7 batches.

Payoffs in a game of chance

  • Consider a game of chance where it is possible to lose $1.00, break even, win $3.00, or win $5.00 each time one plays.

  • The probability distribution for each outcome is as follows:

Outcome [\(x\)] -$1.00 $0.00 $3.00 $5.00
Probability [\(P(x)\)] 0.3 0.4 0.2 0.1
  • The mean for this game:

\[\sum_{x} x P(x)=-\$1\times 0.3 + \$0\times 0.4 + \$3\times 0.2 + \$5\times 0.1\] \[= -\$0.30 + \$0.60 + \$0.50 = \$0.80\]

  • In the long run, expect to win about 80 cents.

Exercise: Textbook revenue

Two books are assigned for a statistics class: a textbook (CAD 137) and a study guide (CAD 33). The probability that a student buys neither is 0.20, the textbook only is 0.55, and both is 0.25.

  • Question (2 min)
  • Answers
  1. Write down the probability distribution for revenue per student.
  2. What is the expected revenue per student?
  3. What is the expected revenue for a class with enrollment 100?

  • Let \(X\) denote revenue per student. It takes the values CAD 0, CAD 33, CAD 137, and CAD 170, i.e. \(X \in \{0,33,137,170\}.\)

  • The probability distribution is:

\(x\) \(P(x)\)
0 0.2
33 0
137 0.55
170 0.25

  • Because \(0.2 + 0.55 + 0.25 = 1\), the probability of buying the study guide only must be zero.

  • Expected revenue per student:

\[0 \times 0.2 + 137 \times 0.55 + 170 \times 0.25 = 117.85.\]

  • Expected revenue for enrollment 100: \(100 \times 117.85 = 11{,}785\) CAD.

Variance of a discrete RV

  • The variance of a discrete random variable \(X\) is the number \[\sigma^2 = \sum_x (x-\mu)^2 P(x).\]
  • Variance measures how spread out \(X\) is around its mean \(\mu\)
    • each squared deviation \((x-\mu)^2\) is weighted by its probability
    • large deviations from \(\mu\) contribute more to variance
  • The standard deviation is \(\sigma = \sqrt{\sigma^2}\) (same units as \(X\))
  • Alternative formula (often easier to compute): \[\sigma^2 = \left(\sum_x x^2 P(x)\right) - \mu^2 = E(X^2) - [E(X)]^2\]
  • Interpretation:
    • \(\sigma^2\) and \(\sigma\) measure how variable \(X\) is across repeated trials
    • if \(X\) is in dollars, \(\sigma^2\) is in “dollars squared” and \(\sigma\) is in dollars
    • \(\sigma^2 \geq 0\) always (it is a sum of non-negative terms)

Investment payoffs: same mean, different variance

  • Two investments, each with equal probability of two payoffs:
    • Option A: $40 or $60
    • Option B: $0 or $100
  • Both have \(E(X) = 50\), but Option B is much more variable.
  • Variance captures this difference.

Fair coin variance

  • Recall: \(P(X=0) = P(X=1) = 1/2\), with \(\mu = 0.5\).

  • Using the definition: \((0-0.5)^2 = (1-0.5)^2 = 1/4\), so \[\sigma^2 = 1/4 \times 1/2 + 1/4 \times 1/2 = 1/4.\]

  • Using the alternative formula: \(E(X^2) = 0^2 \times 1/2 + 1^2 \times 1/2 = 0.5\), so \[\sigma^2 = E(X^2) - \mu^2 = 0.5 - 0.25 = 0.25.\]

Exercise: Variance of a symmetric RV

Consider a discrete RV with \(x \in \{-1, 1\}\) and \(P(x) = 1/2\) for each value.

  • Question (2 min)
  • Answers
  • Compute the variance.

  • The mean is \(\mu = -1 \times 1/2 + 1 \times 1/2 = 0\).

  • For \(x=-1\): \((-1 -0)^2 = 1\). For \(x=1\): \((1-0)^2=1\).

  • Therefore, \[\sigma^2 = 1/2 \times 1 + 1/2 \times 1 = 1.\]

Exercise: Variance of a shifted RV

Consider a discrete RV with \(x \in \{4, 6\}\) and \(P(x) = 1/2\) for each value.

  • Question (2 min)
  • Answers
  1. What is the relationship between this RV and the one from the previous exercise?
  2. Compute the variance.
  • This RV is a shifted version: add 5 to each value of the previous RV.
  • The mean is \(\mu = 4 \times 1/2 + 6 \times 1/2 = 5\).
  • For \(x=4\): \((4-5)^2 = 1\). For \(x=6\): \((6-5)^2 = 1\).
  • Therefore, \(\sigma^2 = 1/2 \times 1 + 1/2 \times 1 = 1\).
  • The variance is the same! Shifting does not change variance.

Exercise: Variance of a scaled RV

Consider a discrete RV with \(x \in \{-2, 2\}\) and \(P(x) = 1/2\) for each value.

  • Question (2 min)
  • Answers
  1. What is the relationship between this RV and the symmetric RV from two exercises ago?
  2. Compute the variance.
  • This RV is a scaled version: multiply each value of the original by 2.
  • The mean is \(\mu = -2 \times 1/2 + 2 \times 1/2 = 0\).
  • For \(x=-2\): \((-2-0)^2 = 4\). For \(x=2\): \((2-0)^2 = 4\).
  • Therefore, \(\sigma^2 = 1/2 \times 4 + 1/2 \times 4 = 4\).
  • Scaling by a factor \(c\) multiplies the variance by \(c^2\): the original had \(\sigma^2 = 1\), and \(2^2 \times 1 = 4\).

Exercise: Airline baggage fees

An airline charges $25 for the first checked bag and $35 for the second. Among passengers: 54% check no bags, 34% check one bag, and 12% check two bags. Let \(X\) = baggage fee revenue per passenger.

  • Question (2 min)
  • Answers
  1. Write down the probability distribution of \(X\).
  2. Compute the expected baggage revenue per passenger.
  3. For a flight of 120 passengers, what is the expected total baggage revenue?
  • If 0 bags: fee = $0. If 1 bag: fee = $25. If 2 bags: fee = $25 + $35 = $60.
\(x\) 0 25 60
\(P(x)\) 0.54 0.34 0.12
  • \(E(X) = 0 \times 0.54 + 25 \times 0.34 + 60 \times 0.12 = 0 + 8.50 + 7.20 = 15.70\)
  • Expected revenue per passenger: $15.70
  • For 120 passengers: \(120 \times 15.70 = \$1{,}884\)
