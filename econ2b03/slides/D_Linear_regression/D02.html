Part I: The Linear Model

Simple linear regression model

  • Model: \[Y = \beta_0 + \beta_1 x + \epsilon\]
  • Deterministic part (population regression line): \(\beta_0 + \beta_1 x\)
  • Random part (error term): \(\epsilon\)
    • mean 0
    • standard deviation \(\sigma\)
  • Interpretation: \[\mathbb{E}(Y \mid X = x) = \beta_0 + \beta_1 x\]

Linear model picture

Interpretation: conditional mean

For each fixed value of \(x\) a sub-population […] is determined, such as the collection of all houses with 2,100 square feet of living space. For each element of that sub-population there is a measurement \(y\), such as the value of any 2,100-square-foot house. The deterministic part of the model corresponds to the population mean of all the \(y\)-values for each particular value of \(x\).

  • Fix a value of \(x\) (a subpopulation).
  • The deterministic part gives the population mean of \(Y\) in that subpopulation: \[\mathbb{E}(Y \mid X = x) = \beta_0 + \beta_1 x\]
  • Individual observations differ from that mean because of \(\epsilon\).

Part II: Fitting the Line

Goal: estimate the population line

  • In the population, the regression line is determined by \((\beta_0,\beta_1)\).
  • In a sample, we do not know \((\beta_0,\beta_1)\).
  • Goal: use sample data (size \(n\)) to estimate \(\beta_0\) and \(\beta_1\).

Predictions and residuals

  • A candidate line is determined by \((b_0,b_1)\): \[\widehat y = b_0 + b_1 x\]
  • At observation \((x_i, y_i)\):
    • predicted value: \(\widehat y_i = b_0 + b_1 x_i\)
    • residual: \(e_i = y_i - \widehat y_i\)

Residuals in a scatter plot

Residual plots: what to look for

  • A good linear fit leaves residuals with no obvious pattern.
  • Curvature in the residual plot suggests a straight line is not adequate.
  • Large isolated residuals may indicate unusual observations worth checking.

Least squares regression line

  • For a candidate line, the sum of squared errors is: \[SSE(b_0,b_1) = \sum_{i=1}^n (y_i - \widehat y_i)^2\]
  • We pick the line that minimizes SSE.
  • The least squares regression line is: \[\widehat y = \widehat\beta_0 + \widehat\beta_1 x\]
  • It is the line that minimizes SSE over all \((b_0,b_1)\).

Least squares formulas

The least squares line is given by \[\begin{align*} \widehat\beta_1 &= \frac{SS_{xy}}{SS_{xx}} \\ \widehat\beta_0 &= \bar y - \widehat\beta_1 \bar x \end{align*}\] where \[\begin{align*} SS_{xx} &= \sum x^2 - \frac{1}{n}(\sum x)^2 \\ SS_{xy} &= \sum xy - \frac{1}{n}\left(\sum x\right)\left(\sum y\right) \end{align*}\]

  • \(\bar x\) and \(\bar y\) are sample means.
  • The least squares line passes through \((\bar x, \bar y)\).

Least squares intuition (simulation)

  • Adapted from Wickham and Grolemund (2017, Chapter 23).
  • In this demo we use RMSE, a scaled version of SSE:
    • \(\text{RMSE} = \sqrt{SSE/n}\)
    • minimizing RMSE is equivalent to minimizing SSE

Residuals and RMSE

Comparing lines by RMSE

sim1_rmse <- function(a1, a2) {
  rmse(c(a1, a2), sim1)
}

models <- models |>
  mutate(dist = purrr::map2_dbl(a1, a2, sim1_rmse))

ggplot(sim1, aes(x, y)) +
  geom_point(size = 2, colour = "grey30") +
  geom_abline(
    aes(intercept = a1, slope = a2, colour = dist),
    data = filter(models, rank(dist) <= 30)
  )

Minimizing SSE (optional)

  • Least squares chooses \((\widehat\beta_0,\widehat\beta_1)\) to minimize: \[\sum_{i=1}^n (y_i - b_0 - b_1 x_i)^2\]
  • Derivation: see whiteboard

Regression vocabulary

  • The equation \(\widehat y = \widehat\beta_0 + \widehat\beta_1 x\) is the least squares regression equation.
  • Extrapolation: using the line to predict at \(x\) values outside the observed range.
  • The sum of squared errors for the least squares line is: \[SSE = \sum (y - \widehat y)^2\]
    • equivalently: \[SSE = SS_{yy} - \widehat\beta_1 SS_{xy}\]

Plotting the regression line

  • geom_smooth(method = "lm") adds the least squares line to a scatter plot.
  • By default, it also adds a confidence band.
elmhurst |>
  ggplot(aes(x = family_income, y = gift_aid)) +
  geom_point() +
  geom_smooth(method = "lm")

Reading the plot

  • Points above the line have positive residuals:
    • the line under-predicts \(y\) there
  • Points below the line have negative residuals:
    • the line over-predicts \(y\) there
  • The band shows uncertainty in the estimated line (more in Part C).

Turning off the confidence band

  • Add se = FALSE if you only want the fitted line.
elmhurst |>
  ggplot(aes(x = family_income, y = gift_aid)) +
  geom_point() +
  geom_smooth(method = "lm", se = FALSE)

Exercises

Exercise: deterministic vs random part (SZ 10.3)

The simple linear regression model is \(y = \beta_1 x + \beta_0 + \epsilon\).

  • Question (2 min)
  • Answers
  1. Identify the deterministic and random parts.
  2. Is \(\beta_1\) a statistic or a population parameter?
  1. Deterministic: \(\beta_1 x + \beta_0\). Random: \(\epsilon\).
  2. \(\beta_1\) is a population parameter that describes the population regression line.

Exercise: sign of the residual

Suppose a point lies above the fitted regression line.

  • Question (1 min)
  • Answers

Is its residual positive or negative? What does that say about the model’s prediction at that point?

The residual is positive. The observed value is above the fitted value, so the regression line under-predicts \(y\) for that observation.
